# uni-app vue3 模板

## 安装
可以使用`npm` \ `cnpm` \ `yarn` \ `pnpm` 等安装工具。

建议使用`pnpm`

```
pnpm i
```

## 启动

uni-app提供了多种环境的启动方式，可以根据`package.json`的`script`字段，来选择执行

比如编译微信小程序的方式

```
pnpm run dev:mp-weixin
```


## 已经集成的功能
- eslint代码验证，以及保存自动修复
- git-commit 代码提交前验证代码格式
